# more_ds
springboot2 + mybatis + mysql + oracle + sqlserver +  (fake)restful   [ 多数据源配置 ]


### 环境介绍
- web框架：SpringBoot2
- orm框架：Mybatis
- 数据库连接池：Druid
- 主数据源：Mysql
- 从数据源1：Oracle
- 从数据源2：SqlServer2012
- 运行平台：Jdk8
- 日志配置：Logback

### 实现功能
- 多个数据源，动态切换
- swagger-ui提供伪Restful接口文档
- AOP实现统一入参校验
- AOP实现统一异常处理
